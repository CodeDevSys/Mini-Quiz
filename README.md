Mini Quiz Game

 Purpose of the Project
This small Python program is a **Mini Quiz Game** designed to test basic general knowledge.  
It asks the user 5 questions and shows how many were answered correctly.  
The goal is to practice **input/output**, **conditionals**, and **score tracking** in Python.

---

 Technologies Used
- **Language:** Python 3  
- **Tools:** Any text editor or IDE (e.g., VS Code, PyCharm)  
- **Platform:** Command-line interface (CLI)

---

 How to Run
1. Download the file `quiz.py`
2. Open a terminal or command prompt
3. Run the program with:
   
   python quiz.py
